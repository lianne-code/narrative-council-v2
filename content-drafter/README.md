# Content Drafter

Turns a locked Narrative Brief into a first draft. One pen. No committee.

## What it does

The Content Drafter takes a locked brief — from the Brief Builder or your own — and produces a Safety Tax-resistant first draft. It writes from the committed claim. It does not soften, hedge, or introduce committee language.

Every draft includes an editor's note flagging the one line most likely to get softened in review and the structural choice most likely to be challenged — so you know what to defend.

## Content types

| Type | Description |
|---|---|
| Thought Leadership Post (LinkedIn) | 800-1200 words. One argument, fully developed. |
| Campaign Narrative | The strategic story arc — for a creative team, not an ad |
| Positioning Statement | Internal and external versions. Both defensible. |
| Executive Briefing | One page. For a CMO presenting upward. |
| Substack / Newsletter Essay | 1000-1500 words. A genuine argument, not a roundup. |

## Usage

```bash
export ANTHROPIC_API_KEY=your_key_here
python drafter.py
```

You will be prompted to load a saved brief file or paste one directly, then select your content type.

## The One-Pen Principle

A narrative written by committee has no voice. The drafter holds the pen. The CMO will have opportunities to soften the draft in review. The drafter's job is to give them something worth defending.
